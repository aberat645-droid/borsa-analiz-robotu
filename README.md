# Borsa Analiz Robotu

Bu, Streamlit kullanılarak geliştirilmiş canlı bir borsa analiz aracıdır. Hisse senetleri için Bollinger Bantları, RSI ve Basit Hareketli Ortalama (SMA) hesabı yaparak, alım/satım sinyalleri üretir.

## Kurulum ve Çalıştırma

1. Python yüklü olduğundan emin olun.
2. Gerekli kütüphaneleri yükleyin:
   ```bash
   pip install -r requirements.txt
   ```
3. Uygulamayı çalıştırın:
   ```bash
   streamlit run app.py
   ```
